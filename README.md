# World Cup 2026 Interactive Bracket

A static, browser-only interactive World Cup bracket.

## Files

- `index.html`
- `style.css`
- `script.js`

## Publish on GitHub Pages

1. Create a new GitHub repository, for example `world-cup-bracket`.
2. Upload these three files into the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, set:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Click **Save**.
6. Wait about 1–3 minutes. GitHub will give you a public URL.

Anyone who opens the URL can fill out their own bracket. Picks are saved in their own browser.
